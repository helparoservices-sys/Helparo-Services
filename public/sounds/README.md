# Notification Sounds

This folder should contain notification sound files used by the application.

## Required Files

1. **notification.mp3** - Short notification sound for job alerts and messages

## Where to get sounds

You can download free notification sounds from:
- https://freesound.org/ 
- https://mixkit.co/free-sound-effects/notification/
- https://notificationsounds.com/

## Recommended specifications

- Format: MP3
- Duration: 0.5 - 2 seconds
- Size: Under 100KB for fast loading
